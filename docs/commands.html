Superior Bot - List of Commands

Prefix - @mention

General Commands

  • ping - Returns Bot latency

  • avatar - Send the messager's avatar in dm

  • guildinfo - Info about the guild

  • userinfo @[member] - Info about pinged member

  • embed - Generate a Beautiful embed for you

  • bitcoin - Shows current bitcoin price in $

  • serverstats - Current Stats of your server

  • welcomer - Setup welcomer for your server

  • joined @[member] - Shows the member joined on tye guild

  • uptime - Shows the uptime of the bot

  • botinfo - Info about the bot

    • Mathematics Commands

  • add [num1][num2] - Add the given numbers(the number must be integer)

  • subtract [num1][num2] - Subtract the given numbers(the number must be integer)

  • divide [num1][num2] - Divide's the given numbers(the number must be integer)

  • multiply [num1][num2] - Add the given numbers(the number must be integer)

    • Fun Commands

  • meme - Sends a meme

  • slap [@member] - Slaps a pinged member

  • mentionme - Mention you

  • dice - Rolls the dice

  • toss - Possible Responces (heads, tails), puts toss

  • meow - Sends a random cat image

  • hug [@member] - Hugs a pinged member

    • Mod Commands

  • purge [limit] - Deletes the given number of messages(the limit must be a integer and the max limit is 2000). This command requires Manage-Guild permission

  • kick [@member] - Kick the pinged member(this command requires Manage-Guild permission)

  • ban [@member] - Ban the pinged member(this command requires Manage-Guild permission)

  • unban [previously_banned_member] - Unban the member from banned list(this command requires Manage-Guild permission)

  • mute [@member] - Mute the pinged member(this command requires Manage-Guild permission)

  • unmute [previously_muted_member] - Unmutes the previously muted member(this command requires Manage-Guild permission)

  • nuke - Nuke the current channel(this command requires Manage-Guild permission)

    • Search Commands

  • google [search] - Search google for the given query

  • youtube [search] - Search for YT videos

  • yahoo [saerch] - Search yahoo for the given query

    • Image Fun Commands

  • calling [text] - Calling image with given text

  • achievement [text] - Achievement image with given text

  • captcha [text] - Captcha image with given text

  • facts - Facts image with given text

  • challenge - Challenge image with given text

  • scroll - Scroll image with given text

    • Text Fun Commands

  • reverse [text] - Reverses the given text(text must be a string)

  • echo [text] - Repeats the text(text must be a string)

  • randomnum - Sends a random number

  • greentext [text] - Makes the given text in green(text must be a string)

  • bluetext [text] - Makes the text in blue(text must be a string)
